# Data

This folder contains the dataset used for the **IEEE-CIS Fraud Detection Project**.

⚠️ The dataset is not included in this repository because it is too large.

## Dataset Source

IEEE-CIS Fraud Detection Dataset
https://www.kaggle.com/competitions/ieee-fraud-detection/data

## Instructions

1. Download the dataset from the Kaggle link above.
2. Extract the files.
3. Place the following files inside this **data/** folder:

* train_transaction.csv
* train_identity.csv
* test_transaction.csv
* test_identity.csv

After placing the dataset in this folder, you can run the notebooks normally.
Dataset not included due to privacy
